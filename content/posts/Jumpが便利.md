最近使い始めた[Jump](https://github.com/gsamokovarov/jump)コマンドが便利という話。Rebuild.fmの[#313](https://rebuild.fm/313/)を聴いていて初めてJumpコマンドについて知りました。

# インストール
macOSであれば
```shell
brew install jump
```
でインストールですることができる。その他の詳しいセットアップは[Integration](https://github.com/gsamokovarov/jump#integration))の項目を参照してください。

# 機能
インストール後、デフォルト設定の場合`j something`で`something`にファジーマッチする使用頻度の高いディレクトリに移動してくれる。新しいディレクトリ等は一度`cd`コマンドなどで一度そのディレクトリまで移動しないと候補としてカウントされないので注意が必要です。
